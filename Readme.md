# Contributions

Pour ce projet, mes contributions sont :

- **Mouvement du joueur**  
  Implémentation des contrôles pour le joueur afin d'assurer une jouabilité fluide.

- **Implémentation des sbires**  
  Ajout d'un ennemi avec une zone d'aggro et une zone d'attaque : il poursuit le joueur lorsqu'il entre dans son champ de détection et l'attaque à portée.

  Mise en place d’un système de collision robuste entre le joueur et les ennemis pour une interaction cohérente avec l’environnement.

- *Création d’objets pour le boss**

## Jeux D'Enigme mobile
Repo : [https://github.com/MtPoison/DepthsOfHue](https://github.com/MtPoison/DepthsOfHue)


## Mes Contributions

- **Système de sauvegarde des données**  
  Mise en place d’un système de sauvegarde et de chargement à partir d’un fichier JSON, avec la possibilité de gérer toutes les catégories ou une seule spécifique.  
  Des sécurités ont été intégrées pour éviter les erreurs en cas de fichier vide ou de données manquantes.

- **Système d’inventaire complet**  
  Conception d’un système d’inventaire permettant d’ajouter ou de retirer des objets.  
  Utilisation de ScriptableObjects pour définir les objets stockables, avec la gestion des quantités multiples selon le type d’objet.

- **Interface utilisateur de l’inventaire**  
  Création d’une fenêtre contextuelle affichant les objets de l’inventaire avec des descriptions, anticipant les évolutions futures de l’interface.

- **Modes de jeu "Pairs" et "Pillars"**  
  Conception et implémentation de deux modes de jeu originaux :  
  - Mode Pairs : associer des éléments.  
  - Mode Pillars : interagir avec des piliers selon des règles définies.  
  Importation et intégration de toutes les textures associées.

- **Développement d’un shader pour bulles**  
  Création d’un shader visuel dynamique pour les effets de bulles, apportant une touche esthétique immersive.

- **Système de zoom caméra**  
  Mise en place d’un zoom caméra adaptatif pour améliorer la lisibilité et le ressenti lors du gameplay.

- **Paramètres du jeu adaptatifs**  
  Tous les paramètres du jeu ont été rendus responsives, assurant une compatibilité parfaite avec différentes résolutions d’écran.  
  Intégration de tous les assets nécessaires dans le menu des paramètres.

## Fête Foraine Unreal Bp

Repo : [https://github.com/MtPoison/FeteForaineUnreal](https://github.com/MtPoison/FeteForaineUnreal)


## Contenue du projet

- **Jeu de Chamboule tout**  
  Conception d'un jeu où le joueur dispose de quatre tentatives pour renverser toutes les canettes, avec un système de score basé sur la réussite des tirs.

- **Système de auto-tamponneuse**  
  Création d'un système où le joueur doit éliminer toutes les voitures le plus rapidement possible avant de quitter l'aire de jeu. Un défi de vitesse et de coordination.

- **Montagnes russes**  
  Conception d’une petite montagne russe pour enrichir l’expérience du joueur, avec des effets visuels et sonores pour augmenter l'immersion.

- **Navigation des NPCs avec NavMesh**  
  Utilisation de NavMesh pour animer les NPCs, qui réagissent à la présence du joueur en le saluant lorsqu'il s'approche.

- **Système d'IA de monstre**  
  Un monstre rôde dans l'environnement et, si le joueur s'en approche trop, il commence à le poursuivre et à l'attaquer, ajoutant un élément de suspense et de danger.
